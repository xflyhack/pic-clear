# 自动化 bat 脚本

给堡垒机 Windows 用的一键脚本。把两个 exe 放到系统 PATH（推荐 `C:\Windows\System32`），bat 自动定位 `Z:\sjbz_*` 源目录，让你**挑选要处理的子目录**，把结果写到 `Z:\切帧结果\` 下、层级完全对齐。

## 前置准备（一次性）

1. 把 `extract_frames.exe`、`dedupe_pic.exe`、`license.lic` 三个文件复制到 `C:\Windows\System32\`
   - 也可以放到别的目录，然后把该目录加到 `PATH`
   - **`license.lic` 必须和 exe 同目录**
2. 把这三个 bat 复制到桌面（或任何顺手的地方）：
   - `run_all.bat` —— 一键跑完抽帧+去重
   - `step1_extract.bat` —— 只抽帧
   - `step2_dedupe.bat` —— 只去重
3. 建议永久关闭 cmd 的"快速编辑模式"（避免鼠标点击导致卡住）：
   右键 cmd 窗口标题栏 → 属性 → 编辑选项 → 取消勾选 **快速编辑模式**。

## 使用流程

双击任一 bat 后：

1. **自动定位 sjbz 根目录**
   `Z:\` 下有唯一 `sjbz_*` → 直接用；多个 → 列表选一个；没有 → 让你手工输入路径。

2. **列出根目录下的一级子目录，让你挑**
   ```
   [子目录] Z:\sjbz_20260708 下的一级子目录（共 4 个）：
      [1] 1
      [2] 2
      [3] 3
      [4] 4

   输入方式（可组合）：
      - 序号列表（逗号分隔）：  1,2
      - 序号区间：              1-3
      - 全部：                  all
   请输入你要处理的子目录:
   ```
   例：输入 `1,2` → 只跑前两个子目录。

3. **依次处理每个已选子目录**
   输出目录完全对齐层级：
   ```
   源：      Z:\sjbz_20260708\1\...
   目标：    Z:\切帧结果\sjbz_20260708\1\...
   ```

4. **`run_all.bat` 每处理完一个子目录，会问一次**：
   ```
   Y=真删本目录
   N=跳过本目录（保留报告不删）
   A=对剩余全部子目录一律真删（省得每个都按 Y）
   ```

## 需要改盘符 / 目录前缀 / 输出根？

打开 bat，改开头三行：

```bat
set "DATA_DRIVE=Z:"
set "OUT_ROOT=%DATA_DRIVE%\切帧结果"
set "DATA_PREFIX=sjbz_"
```

## 常见问题

- **窗口卡住不动**：cmd 快速编辑模式，按一下 Enter 或 Esc 恢复；或按上文永久关闭。
- **提示"找不到 exe"**：exe 不在 PATH。要么放 `C:\Windows\System32`，要么改 bat 里 exe 为绝对路径。
- **license.lic 报错**：license 必须和 exe 在同一目录。
- **想改抽帧参数（fps / 扩展名）**：改 bat 里 `extract_frames.exe ... --fps 1 --ext .h265`。
- **想改相似阈值**：改 `--threshold 3`，值越小越严格。
- **子目录名含空格 / 中文**：脚本已用 `""` 包裹路径，直接用即可。
